# Feather Rift Survivors (Open-Source Powered)

A polished arcade survivor game that **works on phone + desktop**.

## Open-source repos used
- Phaser 3 (engine): https://github.com/phaserjs/phaser
- Howler.js (audio): https://github.com/goldfire/howler.js

## Gameplay
- Survive endless enemy swarms
- Auto-fire + dash movement
- XP gems + level-up upgrade choices
- Build synergies (damage, fire rate, magnet, HP, speed)
- Mobile touch controls included

## Controls
- Desktop: WASD/Arrows move, Shift dash
- Mobile: virtual joystick + Dash button

## Deployment
Static site for GitHub Pages (`index.html`, `game.js`).
